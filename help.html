Add an Inventory Item

  • This feature allows you to keep track of stock on hand by adding inventory records into our remote database.
  • To use this feature; click on it in the menu on the left and fill out the item information (name of the item, its category, price, and quantity) and then click Submit.
  • This will send the inventory information into the server for later viewing.

Add a Sale Record

  • This feature allows you to keep track of sales by adding the records into our remote database.
  • To use this feature; click on it in the menu on the left and before inputting data check the table that shows current inventory (retreived from the remote server) to ensure the sale you add is an item you have in stock. Then fill out the sale information (inventory item number, date of sale, quantity sold) and then click Submit.
  • This will send the sale record into the server for later viewing.

Generate a Report

  • This feature allows you to create reports showing sales between two dates, gathered from data from the remote server.
  • To use this feature; click on it in the menu on the left and select the two dates between which you wish to see sales data and then click Submit.
  • This will retreive the sales data for you between the dates you chose and allow you to create a CSV (comma separated value) file with that data by clicking on the Export button.
